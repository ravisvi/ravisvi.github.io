Customer Service Bot

As part of NLP elective

Built Customer Service Representative Bot which could accept and answer various English queries ranging from price of a particular model to feature comparison between two devices. This was done on a corpus generated by the students of PESIT.
We Peformed Named Entity Recognition on input sentences, Relationship Extraction using Machine Learning on obtained named entities and used Natural Language Generation to generate responses for the bot.

Location Prediction

Center for Cloud and Big Data Lab, PESIT

Here I worked on a research project which aimed at predicting location of a user by using just the data which was available on the user's phone. We were provided with a huge corpus of data by Nokia.
We experimented with Hidden Markov Model and trained the system to understand patterns in usage of applications to guess a context. We then analyzed and formulated a way in which we could use this data to predict the next location for the user. This information was then to be used to serve context based advertisements.
